Деревоподібна структура яка змінюється  динамічно при додаванні нового елемента, редагування, чи видалення елемента. Написано на чистому PHP з використанням бази даних MySQL.


Оскільки у вас нема бази данних, яку я створив для виконання цього завдання, 
то я експортував її в репозиторій на github з назвою company.sql 
Для того щоб здійснити підключення згідно мого коду, використовуйте наступні параметри, 
або ж можна внести певні зміни в файл connection.php


$host = 'localhost'; // адреса сервера(локальний сервер)
$user = 'root'; // і'мя користувача

$database = 'company'; //і'мя имя бази даних
